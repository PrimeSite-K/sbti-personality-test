## 🌐 Live Demo

[View Demo](https://sbti-personality-test.vercel.app)

## ✨ Features

- **30 Questions** - Comprehensive personality assessment
- **27 Personality Types** - From CTRL to DRUNK, discover your type
- **15 Dimensions** - S, E, A, Ac, So with 3 sub-dimensions each
- **5 Languages** - English, 中文, हिन्दी, Русский, 日本語
- **Dark Tech Style** - Beautiful glassmorphism design
- **Radar Chart** - Visualize your dimension scores
- **Social Share** - Share your result with friends

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/PrimeSite-K/sbti-personality-test.git
cd sbti-personality-test

# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3000
```

## 📊 Dimensions

| Category | Dimensions | Description |
|----------|------------|-------------|
| **S** - Self | S1, S2, S3 | Self-worth, Self-awareness, Self-drive |
| **E** - Emotion | E1, E2, E3 | Security, Emotional investment, Personal space |
| **A** - Attitude | A1, A2, A3 | Trust, Conformity, Purpose |
| **Ac** - Action | Ac1, Ac2, Ac3 | Achievement, Decisiveness, Planning |
| **So** - Social | So1, So2, So3 | Openness, Intimacy, Authenticity |

## 🎭 Personality Types

27 unique types including: CTRL, BOSS, LOVE-R, MONK, FAKE, GOGO, SOLO, and more...

## 🛠 Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Animation**: Framer Motion
- **i18n**: Custom with localStorage persistence
